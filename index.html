☕ JSONBin.io Setup — 3 steps
1
Create a free account
Open jsonbin.io in a new tab. Click Sign Up (top right). Use your email or sign in with Google.
2
Get your API Key
After signing in, click your profile icon (top right) → API Keys.
Click + Create Access Key. Give it a name like "EVAR Registry". Copy the key that starts with $2b$...
3
Create a Bin (storage container)
In JSONBin, click + Create Bin (left sidebar or homepage).
In the editor, type [] (just two square brackets).
Make sure Private is selected. Click Create Bin.
The URL will change to something like https://jsonbin.io/b/6849...
The part after /b/ is your Bin ID.
📊 Google Sheets Setup
This requires running a small script inside Google Apps Script. Follow every step exactly.
1
Open Code_gs.js in a text editor
Open the downloaded Code_gs.js file in Notepad (Windows) or TextEdit (Mac). Press Ctrl+A then Ctrl+C to copy all text.
2
Open Apps Script from your Google Sheet
In your Google Sheet: click Extensions menu → Apps Script. A new tab opens showing a code editor with function myFunction() {}
3
Replace the code
Click inside the white code area. Press Ctrl+A to select all → press Ctrl+V to paste → press Ctrl+S to save.
4
Deploy → New deployment
Click the blue Deploy button (top right) → New deployment.
Click ⚙ gear icon → Web app.
Set Execute as: Me and Who has access: Anyone → click Deploy.
5
Authorise (the confusing step)
Click Authorise access → choose your Google account.
If you see "Google hasn't verified this app":
  Click Advanced → Go to [app] (unsafe) → Allow
This is your own script — it is safe.
6
Copy the Web App URL and paste here
After authorising, copy the URL ending in /exec
